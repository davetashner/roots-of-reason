# Bronze Age Technology Art — AI Image Generation Prompts

## Goal

Generate **10 technology illustration images** for all Bronze Age (Age 1) technologies in the Roots of Reason tech tree. Each image appears in the tech detail panel (large, ~256x256 display) and as a thumbnail on tech grid buttons (~48x48).

## Art Style Guide

Match the established Stone Age tech art style exactly (see `assets/sprites/tech/` for references):

| Property | Value |
|----------|-------|
| Resolution | **1024x1024 px** |
| Format | PNG with **transparent background** (alpha channel) |
| Style | **Hand-painted digital illustration** — visible brushwork, NOT 3D render or pixel art |
| Palette | Warm earthy tones: browns, tans, ochre, sandy yellows, muted stone grays, with **bronze/copper metallic accents** for this age |
| Composition | **Still-life vignette** — objects arranged on a small patch of ground/surface, viewed from isometric 3/4 top-down (~35 degrees) |
| Ground | Thin natural ground plane (dirt, stone, grass) that fades to transparency at edges — no hard borders |
| Lighting | Warm, slightly directional from upper-left, soft shadows |
| Detail level | High enough to read clearly at 256px, iconic enough to recognize at 48px |
| Text | **None** — no labels, letters, or words in any image |

**Key style notes from existing art:**
- Objects feel tactile and grounded — real weight and texture
- Compositions are tight — 2-4 key objects filling the frame
- Color temperature is consistently warm (no cool blues or grays)
- Edge treatment: objects blend naturally into the ground patch, ground fades to transparent

## Post-Processing

After generating each image:

```bash
# Save source art to tech sprites directory
# Filename must match tech ID exactly
cp <generated_image> assets/sprites/tech/<tech_id>.png

# Verify dimensions
python3 -c "from PIL import Image; img=Image.open('assets/sprites/tech/<tech_id>.png'); print(img.size, img.mode)"
# Expected: (1024, 1024) RGBA
```
